# [MirBot version 1.0]


## Установка
```
$ git clone https://github.com/StasikLeyshin/mir_bot.git
$ cd mir_bot
$ pip install -r requirements.txt
```

Создайте текстовый файл `tokens.txt`</br>
И положите в него токены от групп вк в формате `токен&тема`</br></br>
## Запуск
```
$ python main.py
```
